Site One Page

Ideal para negócios de pequeno ou médio porte, o site One Page é um site onde você inclui todas as informações que o visitante precisa de forma objetiva e criativa em uma única página.

Wireframe

Wireframe é uma representação visual da estrutura básica de uma página. O ideal, é que ele seja feito da maneira mais simples possível, mostrando apenas o essencial como uma espécie de rascunho, sem cores ou imagens. Clique no botão ou escaneie o QR Code para obter o Wireframe desta página.
